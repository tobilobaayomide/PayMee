# Paymee

Paymee is a modern, mobile-first financial dashboard and management platform.

## Tech Stack
- Next.js (React, TypeScript)
- Tailwind CSS
- Supabase (backend & authentication)
- Google Fonts: Josefin Sans (headers), Raleway (body)

## Features
- Dashboard overview, analytics, and quick actions
- Card and transaction management
- Profile, security, and notification settings
- Highly responsive, compact UI for mobile
